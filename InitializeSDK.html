Initializing the iOS SDK

In your AppDelegate.m file, import the FranklySDK.h: 


	#import “AppDelegate.h”
	#import “FranklySDK/headers/FranklySDK.h”

In the AppDelegate.m file under the application:didFinishLaunchingWithOptions method, add the following: 


	(BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions {

	// serverType is one of the following FRSDKServerType enum values
	//
    //		FRSDKServerTypeTest
    //		FRSDKServerTypeProduction
	//
	[FranklySDK initializeSDKWithServerType: FRSDKServerTypeProduction];

	return YES;

	}

Initializing the Android SDK

The Frankly SDK requires the following permissions: 


android.permission.INTERNET
android.permission.ACCESS_NETWORK_STATE
android.permission.READ_EXTERNAL_STORAGE

Partners must declare the Frankly Android SDK’s content provider with their app’s package name as a prefix of the authorities value in the AndroidManifest.xml file, where <app_package_name> is the package name of the app where the Frankly Android SDK is being integrated: 


<provider
    android:authorities="<app_package_name>.contentprovider.FranklySdkContentProvider"
    android:name="com.frankly.franklysdk.contentprovider.FranklySdkContentProvider" />

And RoomListActivity should be declared where <room_list_title> is the title to show in RoomListActivity. 


<activity
    android:name="com.frankly.franklysdk.app.RoomListActivity"
    android:theme="@style/FranklySdk.Theme"
    android:configChanges="keyboardHidden|orientation|screenSize"
    android:label="<room_list_title>"
    android:screenOrientation="portrait" />

Initializing the JavaScript SDK

All operations to the API are made from instances of the FranklyClient class. Those objects expose methods to the application which map to remote procedure calls (RPCs) on the Frankly servers and they also negotiate and maintain the state required by the API's security policies.

Here's how FranklyClient instances are created: 


	var FranklyClient = require( 'frankly-js' )

	var client = new FranklyClient()
